# Выполнение лабораторной работы

**Задача разложения числа на множители**

Разложение составного числа на множители -- одна из первых задач, использованных для построения криптосистем с открытым ключом.
Пусть дано положительное целое число

$$
n = p_1^{a_1} p_2^{a_2} \dots p_s^{a_s},
$$

где $(p_i)$ — различные простые числа, $(a_i \ge 1)$.
На практике достаточно найти нетривиальное разложение:

$$
n = pq,\qquad 1 \le p \le q < n.
$$

```Julia
n = 1359331
c = 1
a = c
b = c
d = 1
function f(x)
    return (x^2 + 5)%n
end
while d == 1
    a = f(a)
    b = f(f(b))
    d = gcd(abs(a-b),n)
end
```

В результате выполнения кода мы найдем делитель: 1181, что совпадает с примером в лабораторной работе ([рис. @fig-001]).
