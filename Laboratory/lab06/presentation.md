# Цель работы

Изучать задачу разложение числа на множители и реализовать алгоритм p-метод Полларда.

# Задание

1. Изучить задачу разложение числа на множители.
2. Реализовать на Julia алгоритм p-метода Полларда.

# Выполнение лабораторной работы

```Julia
n = 1359331
c = 1
a = c
b = c
d = 1
function f(x)
    return (x^2 + 5)%n
end
while d == 1
    a = f(a)
    b = f(f(b))
    d = gcd(abs(a-b),n)
end
```
